
  # Chatbot Dashboard UI Design

  This is a code bundle for Chatbot Dashboard UI Design. The original project is available at https://www.figma.com/design/RR6ao5iydOkh8XvMQx96j2/Chatbot-Dashboard-UI-Design.

  ## Running the code
  Don't forget to Download Node.js !!!!
  

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the Dashboard 

  To run the fastapi : 
  Run `cd backend`  ==> to be in app.py
  Run `python -m uvicorn app:app --reload` ==> lancer serveur fastapi
